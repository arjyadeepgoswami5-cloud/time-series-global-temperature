# Time Series Analysis of Global Temperature Data

## Objective
To analyze global temperature trends using time-series techniques.

## Dataset
Monthly global temperature anomaly data from multiple sources.

## Analysis Performed
- Data loading and inspection
- Date-time conversion and sorting
- Feature extraction (Year, Month)
- 12-month moving average
- Last 20 years filtering
- Aggregation by source, year, and month
- Pivot table for seasonal analysis

## Tools Used
- Python
- Pandas
- Google Colab

## Conclusion
The analysis reveals long-term warming trends and clear seasonal patterns
in global temperature anomalies.
